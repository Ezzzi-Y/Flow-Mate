# FlowMate 学生版 MVP PRD（简版）

## 1. 目标与范围
- 目标：帮助学生明确“今天做什么/怎么做”，形成清晰的计划与执行。
- 范围：个人学生场景；不支持复杂职场/企业功能。
- 核心闭环：登录 -> 创建任务 -> Agent 计划 -> 今日面板 -> 提醒 -> 完成 -> 复盘。

## 2. 目标用户
- 学生（中学/大学/研究生）。
- 痛点：任务零散、优先级不清、拆解不足、执行不明确。
- 不在App中明确说明面向学生，所有人群均可使用，但我们的功能只满足大学生即可。

## 3. 关键用户故事（MVP）
- 作为学生，我可以用 Google/邮箱登录并查看 AI 额度。
- 作为学生，我可以创建任务并设置截止时间和优先级。
- 作为学生，我可以让 Agent 拆解任务并生成今日计划。
- 作为学生，我能在今日面板看到必做与建议时间段。
- 作为学生，我能收到提醒并标记完成。
- 作为学生，我能查看简要的每日复盘。

## 4. 功能清单与优先级
### P0 必须有
- 账号与鉴权：Google/邮箱登录、会话管理、额度展示。
- 任务清单：新增/编辑/删除/完成，优先级、截止时间、标签。
- 今日面板：必做、过期、建议时间段。
- Agent：生成今日计划、任务拆解、每日复盘。
- 通知：到期提醒、每日计划提醒。
- 数据：本地存储 + 账号同步（基础冲突策略：最近更新优先）。

### P1 应该有
- 搜索与筛选（标签/状态/优先级）。
- 周报汇总与完成率趋势。
- 快捷入口（小组件/快捷创建）。

### P2 可选（后续）
- 课程表与考试周期管理。
- 基于历史的个性化学习建议。
- 更完善的离线冲突处理。

## 5. 主要页面
- 登录
- 今日面板（首页）
- 任务清单
- 任务详情/编辑
- Agent 对话
- 复盘/统计
- 账号与额度

## 6. 核心流程
### 新用户
- 登录 -> 基础设置（学习偏好/时间段）-> 创建/导入任务 -> 生成今日计划 -> 执行与提醒 -> 复盘。

### 日常使用
- 打开今日面板 -> 执行/完成任务 -> Agent 复盘 -> 同步。

## 7. 业务规则
- 任务必填：标题；可选：截止时间、优先级、标签。
- 过期任务在今日面板优先展示。
- Agent 仅可读写任务与日程，不接触邮件/联系人。

## 8. 非功能需求
- 性能：首页加载 < 2s，列表交互顺滑。
- 可用性：离线可读写，联网自动同步。
- 隐私：提供数据导出/删除入口与基础隐私说明。

## 9. 里程碑（建议）
- M1（2-3 周）：账号 + 任务清单 + 今日面板。
- M2（2-3 周）：Agent + 通知 + 基础同步。
- M3（2 周）：复盘/统计 + 搜索 + 小组件。
